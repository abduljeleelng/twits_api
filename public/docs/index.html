    Introduction

    documentation for the Twits API

    This documentation aims to provide all the information you need to work with our API.

    As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

    Base URL

    https://tweetis-api.herokuapp.com

    Authenticating requests

    Authenticate requests to this API's endpoints by sending an Authorization header with the value "Bearer {token}".

    All authenticated endpoints are marked with a requires authentication badge in the documentation below.

    You can retrieve your token by login with your email and password, token will come along with your details.

    Auth User management

    APIs for authenticating users

    POST api/auth/signup

    Example request:
    curl --request POST \
    "https://tweetis-api.herokuapp.com/api/auth/signup" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"oberbrunner.izabella@example.org\",
    \"password\": \"e\",
    \"password_confirmation\": \"id\"
}"

    const url = new URL(
    "https://tweetis-api.herokuapp.com/api/auth/signup"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "oberbrunner.izabella@example.org",
    "password": "e",
    "password_confirmation": "id"
}

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
    $client = new \GuzzleHttp\Client();

    'https://tweetis-api.herokuapp.com/api/auth/signup',
    [
        'headers' => [
            'Accept' => 'application/json',
        ],
        'json' => [
            'email' => 'oberbrunner.izabella@example.org',
            'password' => 'e',
            'password_confirmation' => 'id',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
